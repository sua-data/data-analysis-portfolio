## 🚀 About Me

## 📊 Projects

### 1️⃣ E-commerce Sales Analysis

📁 `projects/01-ecommerce-analysis`

* 쇼핑몰 데이터를 분석하여 매출 패턴 및 인기 상품 도출
* 월별 매출 추이 및 상품별 판매량 분석

👉 **핵심 경험**

* 데이터 전처리 및 분석 흐름 이해
* 시각화를 통한 인사이트 도출

---

### 2️⃣ Employment News Classifier

📁 `projects/02-employment-news-classifier`

* 뉴스 제목을 기반으로 고용기사 여부를 분류하는 머신러닝 프로젝트
* FastAPI를 활용하여 예측 기능을 웹 서비스로 구현

👉 **핵심 기능**

* 엑셀 데이터 통합 및 전처리
* 다수 라벨러 기반 target 생성
* 머신러닝 모델 학습 및 평가
* 고용기사 / 비고용기사 예측 API
* 데이터 및 예측 결과 시각화

👉 **핵심 경험**

* 데이터 불균형(class imbalance) 문제 해결
* accuracy가 아닌 **recall / f1-score 기반 평가**
* 머신러닝 모델을 실제 서비스 형태로 구현
